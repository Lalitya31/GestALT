Heart and Soul

Every decision in this project was made with intentionality. From the perception simulation engine to the typography choices, GestALT is not just a portfolio piece—it's a statement about how learning systems should work.

UI/UX is inherently visual and experiential. Yet most beginners learn it through text-heavy documentation and static explanations. This contradiction never made sense to me.

GestALT emerged from a simple belief: you learn design by experiencing the consequences of your decisions, not by memorizing principles. Perception should be simulated, tested, and refined through interaction.

The perception engine doesn't give subjective opinions—it calculates attention flow, hierarchy strength, and cognitive load using deterministic rules based on visual properties. The accessibility validator enforces WCAG standards in real-time, not as suggestions. The adaptive learning system tracks your patterns and adjusts challenges accordingly.

This fusion of design sensitivity and engineering rigor reflects what I believe software should be: technically sophisticated and deeply human.

GestALT is an alternative because it treats perception as something to be modeled, learning as something to be personalized, and accessibility as something to be enforced. It's built with the same care I'd want in any system I use.

This is where my heart and soul went. Into every calculation, every interaction, every line of code.
